# Azure Terraform DevOps Platform

## Overview
This project demonstrates an enterprise-grade Azure infrastructure provisioned using Terraform and deployed via Azure DevOps CI/CD pipelines.

## Architecture
- Azure Virtual Network
- Modular Terraform design
- Remote state using Azure Storage
- CI/CD using Azure DevOps YAML pipelines

## Key Features
- Environment separation (dev/prod)
- Reusable Terraform modules
- Secure state management
- Manual approval before production apply

## How to Deploy
1. Configure Azure service connection in Azure DevOps
2. Update backend storage values
3. Run pipeline to provision infrastructure

## Skills Demonstrated
Terraform, Azure DevOps, IaC, CI/CD, Azure Networking, Cloud Architecture
